# PLI

Projeto para execução de experimentos com instâncias de grafos, geração de rotulações-L(2,1) e armazenamento de resultados em arquivos `.csv`.

## Descrição da estrutura do projeto

### Instâncias de entrada

* data/
Contém instâncias de grafos no formato .mtx, principalmente grafos aleatórios do tipo Erdős-Rényi.
* NR_DIMACS_cleaned/
Contém instâncias limpas da coleção NR/DIMACS no formato .mtx.
* Harwell-Boeing_cleaned/
Diretório relacionado ao conjunto Harwell-Boeing. O arquivo process.log registra o processamento realizado nesse conjunto.
